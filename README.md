# cyber_mosaique
Cyber-Mosaïque est un framework de résilience numérique conçu pour les étudiants en Haïti. Ce prototype Python fonctionne en mode Zero-Data (hors-ligne) pour pallier les crises d'infrastructure. Il intègre un moteur de diagnostic technique et un quiz interactif basé sur le modèle ARCS, offrant une réponse d'ingénierie locale aux cyber-menaces.
